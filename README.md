# 🕹️ Antonym Attack

This is a fast word game. It looks like an old arcade game.

## 🎮 How to Play
1. **Pick a Level:** Choose a level from 1 to 12. 
2. **Look at the Word:** A big word will show up on the screen.
3. **Find the Opposite:** Pick the word that means the opposite. 
4. **Be Fast:** You are being timed. Try to clear the words fast!
5. **Save Your Score:** Put your 3-letter name on the high score board if you are fast.

## 🛠️ The Code
* This game is made in one single file.
* It uses simple HTML, CSS, and JavaScript.
* It does not save cookies. It does not track you.
* It works with screen readers so everyone can play.
